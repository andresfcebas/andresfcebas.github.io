Main Description

Descripción general del proyecto de modding.

En esta se realiza una descripción de lo que se desea realizar en el proyecto. Aquí se define a grandes rasgos los cambios que deseamos realizar sobre el producto conforme a nuestra idea.

Se pueden tener en cuenta los siguientes ítems:

Concepto del Mod

Aquí se realiza una descripción general del proyecto. Se responden las preguntas:

  • ¿En qué consiste mi idea?
  • ¿Qué es lo que deseo hacer?
Características principales

Se establece en este punto las características que tendrá el producto, para ello se pregunta:

  • ¿Qué hace diferente este mod de otros de que ya existen?
Género al que pertenece

Se realiza una asociación del proyecto a un género dependiendo de la idea y del producto que se moddeará. Diferentes tipos de género son shooters, RPG, estrategia, móviles, aventura, plataformas, habilidad, entre otros tipos.

Maquetas

Dentro de este ítem se realizan abstracciones de diseño mediante maquetas. Algunas de estas abstracciones son la interfaz del juego, paneles, menús, etc.

Conceptos

Se realizan descripciones de los conceptos clave que tendrá nuestro proyecto. Entre ellos personajes, armas, unidades, artefactos del juego, simbología, entre otros.
