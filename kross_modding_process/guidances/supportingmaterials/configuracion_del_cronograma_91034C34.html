Configuración del Cronograma

Para configurar el cronograma se procede a los siguiente:

  • Se realiza una reunión con el equipo de desarrollo para listar los modding tasks obtenidas en tareas previas.
  • Se establecen los tiempos de duración de cada modding task.
  • Posterior al tiempo de duración, se procede a establecer las fechas de inicio y fin de las actividades.
  • Se socializan con el equipo de desarrollo el cronograma para asignar roles y responsabilidades.
