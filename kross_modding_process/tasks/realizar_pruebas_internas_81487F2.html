Task: Realizar Pruebas Internas

Purpose
El propósito de esta tarea es de realizar pruebas al producto con el fin de verificar que no existan posibles inconsistencias y errores que afecten el mod final.

Main Description

Pruebas Internas

Las pruebas internas se realizan con el fin de verificar que las modificaciones realizadas al software no hayan provocado daños o inconsistencias en el producto software objeto de trabajo. Además, las pruebas también se realizan para evaluar que la Modding Task realizada corresponda al objetivo que se estableció en un principio.


Key Considerations
Es importante abarcar la mayor cantidad de escenarios de error posibles, ya que los errores detectados en tiempo de lanzamiento del mod, son mucho más costosos y difíciles de manejar que los detectados en tiempo de desarrollo.
