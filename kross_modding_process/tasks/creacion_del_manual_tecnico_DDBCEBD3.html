El propósito de esta tarea es la realización de un documento que guarda todos los datos del desarrollo del mod.

    Main Description

    Manual Técnico

    El manual técnico reune los avances realizados por el equipo de desarrollo en el transcurso del proyecto. En este manual se colectará toda la información técnica del proyecto en cada Modding Task completada. Dentro de la información técnica se incluyen los datos acerca de la característica del producto que fue cambiada y las posibles consecuencias de este cambio (Para futuras pruebas).

    En esta primera actividad solo se realiza el esqueleto del documento, con el fin de tener una plantilla para consolidar posteriormente los datos obtenidos en la fase de Modding.
