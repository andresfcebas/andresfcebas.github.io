Task: Configurar cronograma
Configurar en un cronograma las "Modding Tasks" conforme a su duración y prioridad
Purpose

El propósito de esta tarea es establecer en un cronograma las tareas de modding con el fin de organizar el tiempo de desarrollo y priorizar las actividades más importantes.

      Main Description

      Cronograma

      El cronograma en el desarrollo del proyecto, permite al grupo de trabajo organizar las "Modding tasks" con el fin de establecer fechas previstas para el comienzo y el final de cada tarea. Lo anterior permite realizar una gestión del tiempo que se dispone, además de priorizar las actividades más importantes en el desarrollo.

      El diagrama de Gantt, es una representación gráfica de un cronograma. Este permite a los miembros del equipo estar al tanto de las tareas pendientes, el tiempo que se dispone y la importancia de cada una.

      En la web existen herramientas software comerciales y libres que facilitan la labor de esta actividad.

      Steps
      Listar las Modding tasks con respecto a su prioridad
      Planear el tiempo requerido para la realización de cada tarea
      Establecer las actividades que se pueden realizar en paralelo
      Identificar actividades que necesitan de otras para comenzar
      Representar las actividades con su duración dentro de un cronograma
