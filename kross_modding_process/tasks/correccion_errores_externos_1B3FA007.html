    Main Description

    Errores Externos

    Posterior al Feedback y las pruebas externas, se procede a realizar las respectivas correcciones de los errores hallados. Dentro de esta tarea se determinan dos caminos a seguir:

    1. Si el error corregido fue grave e impide el funcionamiento del mod, se procede a liberar un parche para la versión actual.
    2. Si el error es leve o estético, se procede a corregir el error para el siguiente lanzamiento del mod.
    Key Considerations
    Los errores hallados dentro de esta tarea pueden llegar a ser delicados en su corrección dada su naturaleza imprevista.
