Task: Creación del Manual de Usuario
Se realiza la creación de un manual que facilita la instalación del mod por parte del usuario final.
Purpose
El propósito de esta tarea es de proveer al usuario final información pertinente para la instalación del mod

    Main Description

    Manual de Usuario

    Posterior a las tareas de modificación, el paso a seguir es proveer mediante un manual, información al usuario con respecto a la instalación, requerimientos especiales y novedades que posee el mod.

    Key Considerations
    La información debe ser muy clara. Se debe intentar explicar paso a paso la instalación y los requerimientos que se puedan requerir con la adquisición del mod.
