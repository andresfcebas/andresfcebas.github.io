Task: Documentar Manual Técnico
Se realiza una inclusión de la información sobre la Modding Task desarrollada previamente
Purpose
El propósito de esta tarea es tener en el manual técnico del mod toda la información correspondiente a las modificaciones realizadas, como referencia para futuros desarrollos en el proyecto

      Main Description

      Manual Técnico

      Dentro del manual técnico del mod se incluyen todos los datos correspondientes a las modificaciones resultado de las tareas de moddeado del proyecto. Esta información es importante, ya que debe funcionar como una referencia para el equipo de modders para las pruebas técnicas del producto, corrección de errores y posible adaptación de nuevas Modding Tasks en el proyecto.
