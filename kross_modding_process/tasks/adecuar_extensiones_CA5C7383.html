El propósito de esta tarea es de preparar a modo de extensión los cambios que se realizaron para acoplar al producto base.

Main Description

Extensiones

Una extensión es un componente software que añade características a un software base. En el caso del proceso del modding, el software base es el producto que se encuentra en modificación y las extensiones son cada uno de los paquetes que contienen los datos de cada cambio del mod final.

En esta tarea se deben empaquetar los cambios que se han realizado al producto, mediante herramientas especializadas, con el fin de permitir al usuario final la fácil instalación de nuestro mod. 

Key Considerations
En comunidades de modding se facilitan las herramientas para producir y gestionar los mods a modo de extensiones, con el fin de que el usuario final de nuestro mod, pueda cargarlo sin ningún problema.
