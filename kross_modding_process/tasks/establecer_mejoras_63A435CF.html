Task: Establecer mejoras
Se realiza una lista de las mejoras a realizar en lanzamientos posteriores.
Purpose
El propósito de esta tarea es la continua mejora del producto ofreciendo cada vez más características y calidad al mod.

    Main Description

    Mejoras

    La continua mejora del proyecto ayuda a que el producto final sea de muy buena calidad, al tiempo que se gana más usuarios para el Mod. Algunas mejoras que se realizan a este tipo de productos son estéticas, rendimiento y expansión de características.
