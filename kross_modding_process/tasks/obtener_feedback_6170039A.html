Main Description

Feedback

El feedback es toda aquella información que provee de los usuarios finales del mod. Esta información tiene gran valor, ya que a partir de esta se pueden realizar mejoras al producto, detección y corrección de errores externos a la fase de construcción e inclusión de características para versiones futuras.

Key Considerations

Dentro de la retroalimentación proveída por los usuarios se debe intentar abstraer la mayor cantidad de información útil para el proyecto.
