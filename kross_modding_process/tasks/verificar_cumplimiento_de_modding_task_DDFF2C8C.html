Task: Verificar Cumplimiento de Modding Task
Se realiza una verificación de cumplimiento con respecto a las Modding Task.
Purpose
El propósito de esta tarea es de verificar que posterior a la tarea "Moddear", se cumpla el objetivo de la modding task que se desarrolló previamente.
Relationships
RolesPrimary Performer:
  • Modder
Additional Performers:
  • Lider de Equipo
Process Usage
  • Kross Modding Process  >  Modding  >  Construcción  >  Verificar Cumplimiento de Modding Task
Main Description

Cumplimiento de objetivos

Es importante verificar el cumplimiento de los Modding Items dentro de cada Modding task, con el fin de verificar el avance del proyecto. Esta tarea se debe repetir luego de completar cada Modding Task. Si existen Modding Items que no se han cumplido, se deben desarrollar o evaluar su desarrollo en otra iteración.

Key Considerations
Es importante verificar el cumplimiento de cada Modding Item al acabar cada Modding Task para dar comienzo a la siguiente.
