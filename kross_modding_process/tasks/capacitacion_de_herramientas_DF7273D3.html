Task: Capacitación de Herramientas
Se realiza un entrenamiento de las herramientas que se usarán en el transcurso del proyecto
Purpose

El propósito de esta tarea es realizar un entrenamiento a los miembros del equipo con las herramientas de trabajo que se de disponen para el desarrollo.

Relationships

Main Description

Herramientas del proyecto

En la etapa de Pre-modding se realizó una obtención de herramientas para desarrollar el producto. Ahora el siguiente paso es capacitarse en estas herramientas para poder trabajar eficientemente con ellas. La web es una gran fuente de información disponible para el aprendizaje. En páginas especializadas es posible acceder a manuales de usuario, videos descriptivos, tutoriales paso a paso y foros que se especializan en brindar información útil para el manejo del software que se dispone.

Steps
Listar las herramientas de desarrollo
Buscar información sobre el manejo general de las herramientas
Formarse sobre el manejo de las herramientas
Compartir a los demás miembros la información obtenida cuando se requiera
Key Considerations
En esta tarea se debe procurar aprender los aspectos generales de las herramientas, ya que las prestaciones más avanzadas se pueden aprender a medida que avanza el proyecto.
