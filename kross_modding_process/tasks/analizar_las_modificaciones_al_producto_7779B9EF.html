Task: Analizar las modificaciones al producto
Se realiza un análisis para determinar qué cambios se harán al producto, generando posteriormente los ítems de modificación.
Purpose

El propósito de esta tarea es analizar y posteriormente generar una lista de los cambios que se le realizarán al producto software

    Main Description

    Cambios o modificaciones al producto

    Los mods se constituyen por un conjunto de cambios que se realizan al producto elegido del proyecto. Dentro de esta tarea se realiza una lluvia de ideas con el equipo de trabajo con el fin de determinar los cambios realizables en el proyecto. Cada modificación parte de una idea socializada con el equipo.

    Posteriormente se discuten cambios realizables dentro del proyecto. En este punto se realiza una clasificación de los cambios, se determinan y eliminan los que no serán realizables y los que se pueden posponer para futuros incrementos o versiones del mod.

    Steps
    Realizar una reunión con el equipo de trabajo
    Realizar una lluvia de ideas para el proyecto
    Crear una lista con las ideas
    Establecer las modificaciones que se realizarán al producto
    Desechar las modificaciones no realizables
    Establecer las modificaciones realizables en versiones futuras del mod

    Key Considerations

    Se deben tener en cuenta y listar todas las ideas del equipo, para luego obtener la mayor cantidad de posibles cambios en el mod. En tareas posteriores se realizará una clasificación de estos cambios.
