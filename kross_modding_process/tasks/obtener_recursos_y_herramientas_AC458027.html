Task: Obtener recursos y herramientas
Se realiza una recolección de los recursos necesarios para la realización del mod y las herramientas necesarias para completar la labor.
Purpose
El propósito de esta tarea es de facilitar al grupo de trabajo los recursos y herramientas necesarios para el desarrollo del mod.

Main Description

Herramientas y Recursos

Luego de obtener la mayoría de recursos que dispone el software a modificar, el siguiente paso es facilitar el acceso a las fuentes de recursos para el desarrollo del mod.

Entre las fuentes de recursos se encuentran:

  • Repositorios propios dentro de estos repositorios se pueden almacenar recursos multimedia y herramientas para tener a mano en el transcurso del proyecto. El almacenamiento en nube facilita mucho la labor para compartir dentro del equipo.
  • Webs de recursos multimedia En la web se pueden encontrar diferentes webs que proporcionan recursos gratuitos para el desarrollo de proyectos multimedia. Estas webs proveen muchos recursos como texturas, audio, animación y gráficos en 2D/3D.
  • Webs del producto en la actualidad muchos productos software apoyan el modding. Así las webs oficiales proveen herramientas para facilitar este fin como SDKs.
  • Foros de la comunidad los foros de la comunidad de modding son una excelente fuente para obtener recursos, herramientas y documentación relevante para el equipo de trabajo.
  • Conocimiento del equipo de trabajo cada miembro del equipo puede poseer su propia fuente y conocimiento de herramientas para el desarrollo. Compartir este conocimiento mejora el desarrollo del mod.

Luego de recolectar la información de las fuentes de recursos y herramientas, el siguiente paso es facilitar todo lo recogido al equipo de trabajo. Lo anterior realizando los siguientes pasos:

  1. Compartir en el equipo los enlaces a las webs.
  2. Descargar e instalar las herramientas de trabajo.
  3. Organizar las descargas en un repositorio común para facilitar el acceso.
  4. Descargar y adjuntar la debida documentación de las herramientas para su manejo.
