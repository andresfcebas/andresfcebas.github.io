Description
Main Description

Documento de Descripción del Proyecto

Este documento recoge los datos del diseño del proyecto y se incluyen:

  • La idea principal del proyecto
  • Las ideas secundarias
  • La información del producto que se modificará
  • Los datos de los miembros del equipo y sus responsabilidades
  • Los objetivos
  • Enlaces de las herramientas y recursos en linea

Key Considerations
El DDP es una herramienta de comunicación entre los miembros del proyecto para comunicar los objetivos a cumplir.
