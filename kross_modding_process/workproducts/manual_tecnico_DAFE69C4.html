Main Description

Manual Técnico

Dentro del manual técnico del mod se incluyen todos los datos correspondientes a las modificaciones resultado de las tareas de moddeado del proyecto. Esta información es importante, ya que debe funcionar como una referencia para el equipo de modders para las pruebas técnicas del producto, correción de errores y posible adaptación de nuevas Modding Tasks en el proyecto.
