Manual de Usuario

El manual de usuario es un documento en el que se consolida la información necesaria para la instalación y el manejo del mod. Dentro de este manual se le describe al usuario los pasos a seguir para poder poner a marchar el mod.

Para el manual se debe tener en cuenta referencias como:

  • La versión del producto que se está manejando.
  • Los requisitos previos del sistema para poder funcionar.
  • Especificar la plataforma en la cual debe correr (PC, Consola, SO...).
  • Si es necesaria alguna herramienta extra que deba ser instalada previamente.

Key Considerations
Se deben tener en cuenta todos los posibles inconvenientes en la instalación. Entre ellos la versión del producto base, las herramientas necesarias para instalar y los requerimientos previos.
