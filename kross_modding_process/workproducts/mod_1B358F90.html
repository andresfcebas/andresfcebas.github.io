Description
Main Description

Mod

El mod es el resultado del proyecto de modding que se está ejecutando. Este es una extensión de software que se conectará al software base del proyecto para funcionar. Cada vez que se completa la fase de "Modding", el resultado es la liberación de una nueva versión del mod (Incremento). 

Key Considerations
Al momento de liberar el mod, se debe tener en cuenta todos escenarios de instalación y los inconvenientes que se pueden presentar al usuario.
