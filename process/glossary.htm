D
DDP

Son las siglas del Documento de Descripción del Proyecto

F
Fan-Programador

En el contexto del modding, un fan-programador es un sinónimo de Modder.

M
Mod
Un Mod es una extensión de software, el cual es el resultado del proceso del Modding. Dada su naturaleza de extensión, es necesario poseer el producto software base para ponerlo en funcionamiento .
Moddear
Dentro del Kross Modding Process, es la acción de ejecutar tareas de modificación dentro de un proyecto de modding. Generalmente estas tareas son realizadas por los modders.
Modder
Un modder es la persona encargada de realizar un Mod.
Modding
El modding es la práctica de modificar aspectos y características de un software base. Esta práctica es realizada por personas que se les denomina "Modders" y el producto resultado de la práctica es un "Mod"
Modding Item

El modding item es la descripció de cada una de las modificaciones deseables para el producto final.

Modding Task

Son cada una de las tareas de modificación de software dentro de un proyecto de modding. Este término es usado dentro de Kross Modding Process.

S
SDK

Kit de Desarrollo Estándar Es un kit software que provee herramientas para edición del producto base


T
Tiempo de Liberación
Se refiere al tiempo en que el producto se ha liberado a un repositorio web de modding y se encuentra público para cualquier usuario de la comunidad.
